# filtros-digitais
Implementação de Filtros Digitais (Equalizador de Histograma, Redutores de Ruídos e Detectores de Borda) para a aula de Visão Computacional.

* Equalizador de Histograma
* Redutores de Ruidos
    * Filtro Média
    * Filtro Mediana
    * Filtro Gaussiano
* Detectores de Borda
    * Sobel
    * Prewitt
    * Laplaciano
    * Roberts
    * Laplaciano da Gaussiana
